Pageturner

A teleprompter that actually pays attention
  • Open source solo project

Normal teleprompters scroll text at a constant speed, and don’t adjust if a speaker stops or slows down during their talk. This project is a teleprompter powered by the web speech API, that listens to user’s speech and scrolls the text accordingly.

Visit the Github Repo for details on the algorithm I designed keep track of the user’s process, while dealing with the inaccuracies of speech recognition.
